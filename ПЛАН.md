# Дипломная работа: Исследование систем глубокого обучения для распознавания эмоций людей

## Цель

Разработать прототип мультимодальной системы распознавания эмоций в реальном времени:
- **Видеопоток** → последовательность кадров лица → TCN-классификатор
- **Аудиопоток** → речь → Transformer-классификатор
- **Мультимодальное слияние** → итоговая эмоция

Целевые классы эмоций: `neutral`, `happy`, `sad`, `angry`, `fearful`, `disgusted`, `surprised`  
(7 базовых эмоций по Экману)

## Модуль A: Видео (TCN)

### Почему TCN, а не LSTM/GRU?
- Параллельное обучение (нет рекуррентности)
- Нет проблемы затухания градиентов
- Receptive field явно контролируется глубиной и дилатацией
- Empirically: TCN ≥ LSTM на sequence-задачах (Bai et al., 2018)

### Детальная схема
```
Видео → [MTCNN] → выровненные лица (224×224)
      → [EfficientNet-B0, frozen/fine-tuned] → эмбеддинги (T × 1280)
      → [TCN блоки: dilated causal conv, residual] → (T × 256)
      → [Global Avg Pool или последний шаг] → (256,)
      → [Linear(256, 7)] → softmax → P(emotion)
```

### TCN параметры
- Слои: 4–6 блоков TCN
- Дилатация: 1, 2, 4, 8 (экспоненциальный рост)
- Kernel size: 3 или 5
- Dropout: 0.2
- Длина окна: 16–32 кадра (~0.5–1 сек при 30 fps)

## Модуль B: Аудио (Transformer / wav2vec 2.0)

### Почему Transformer?
- wav2vec 2.0 и HuBERT — SOTA для speech tasks
- Self-supervised предобучение на сотнях часов речи
- Fine-tuning на малых размеченных датасетах даёт высокое качество

### Детальная схема
```
Аудио (raw waveform, 16kHz)
  → [wav2vec 2.0 / HuBERT base] → контекстные эмбеддинги (T' × 768)
  → [Mean pooling по времени] → (768,)
  → [Linear(768, 256) + ReLU + Dropout(0.3)]
  → [Linear(256, 7)] → softmax → P(emotion)
```

### Варианты модели (по сложности)
| Вариант | Модель | Параметры | Качество |
|---------|--------|-----------|---------|
| Лёгкий | wav2vec 2.0 base | 95M | хорошее |
| Средний | HuBERT base | 94M | лучше |
| Тяжёлый | wav2vec 2.0 large | 317M | SOTA |

Рекомендация для диплома: **HuBERT base** с fine-tuning на IEMOCAP.

## Модуль C: Мультимодальное слияние

### Вариант 1 (простой): Взвешенное среднее
```python
p_combined = alpha * p_video + (1 - alpha) * p_audio
```
